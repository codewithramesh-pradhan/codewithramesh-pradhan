## Featured Projects

### 🏥 [Tech Health CDK](https://github.com/codewithramesh-pradhan/tech-health-cdk)
HIPAA-compliant healthcare API infrastructure built with AWS CDK — Cognito authentication, WAF protection, DynamoDB with encryption, KMS key management, CloudWatch monitoring, and CloudTrail audit logging.

### ✈️ [TravelEase Infrastructure CDK](https://github.com/codewithramesh-pradhan/travelease-infra-cdk)
Serverless travel inquiry management system — API Gateway with request validation, Lambda processing, DynamoDB storage, SES email notifications, SQS dead letter queues, and honeypot spam protection.

### 🔐 [AWS IAM CloudFormation](https://github.com/codewithramesh-pradhan/aws-iam-cloudformation)
Enterprise IAM security implementation — role-based access control across 4 groups and 10 users, MFA enforcement, CloudTrail audit logging, permission boundaries, and GitHub Actions CI/CD with Checkov security scanning.
